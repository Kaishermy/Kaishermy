# Escape Room Challenge (Capstone)

My capstone computer science project at Oregon State University was a video game created in Unity based on real-world escape rooms. It was conducted in collaboration with three other developers over the course of three terms from September 2025 to June 2026.
The game includes three separate escape rooms, each with three puzzles and reachable by a menu, which tracks progress across scenes.

**Technologies used:** Unity, C#

**My Contributions:**
- Save/load system that uses JSON files and a persistent data path to facilitate data persistence between scenes and sessions.
- Interaction framework using an interface and Unity's input system to interact with objects in the scene.
- Object pickup system that leverages the customizable interaction interface to carry select objects in front of the main camera.
- Level design, scripting, and select 3D models for the steampunk workshop, one of the three puzzle chambers.
